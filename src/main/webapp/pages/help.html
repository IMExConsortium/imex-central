    4. Comments

      Any registered user can add a public comment to a publication record by visiting Comment tab within Publication Manager page. Comments can address curation issues, data quality or any other issue relevant to the publication. In order to create a new comment fill in the Subject field (if needed, provide more information in the text field below) and click on the 'Add' button. You can use the preview button to see how the comment will be visible to others.

    5. Attachments

      Record owner (submitter) and curators can provide additional information related to the publication record by attaching to it one or more data attachments. The attachments are visible only to the original creator of the record and the curators. Therefore they can be used to exchange, in a database-neutral manner, data files corresponding to partially curated interaction records, information about identifiers of the molecules mentioned in the publication or any other supplementary information that useful during curation process. The attachments can be also used by the original publication authors to provide access to the supplementary experimental data or the initial, self-curated interaction records.

      In order to create a data attachment:

      • Log in. Please, note, that in order to be able to submit attachments you must be the owner (submitter) of the record or your account must have been granted curator status.
      • Navigate to the Attachment tab on the Publication Manager page of the relevant publication.
      • Click on the Browse button and select the file to upload.
      • Select attachment type (currently only plain text, MIF25 and MITAB files are supported) and (optional) one of the attachment flags (MIMIX or IMEX).
      • Optionally, provide short description of the attachment in the subject field
      • Click on the Submit button. The newly created attachmend should appear on the attachment list.
