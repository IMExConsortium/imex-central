    4. Comments

      Any registered user can add a public comment to a publication record by visiting Comment tab within Publication Manager page. Comments can address curation issues, data quality or any other issue relevant to the publication. In order to create a new comment fill in the Subject field (if needed, provide more information in the text field below) and click on the 'Add' button. You can use the preview button to see how the comment will be visible to others.
